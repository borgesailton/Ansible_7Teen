# Ansible_7Teen

Este é um repositório **privado** destinado ao armazenamento e controle pessoal de playbooks do Ansible.

## Objetivo
Centralizar e versionar todos os playbooks e arquivos de configuração (ex: `deploy-ddt36br.yml`) utilizados na automação e deploy de serviços.

## Estrutura
- **Playbooks**: Arquivos `.yml` contendo as definições das automações.
- **Dependências**: Arquivos como `requirements.txt` listando pacotes e coleções necessárias.

---
*Nota: Repositório de uso estrito e exclusivo para controle interno.*
